# ChatDot 商店文案多语言版

## English

### Summary
Add a right-side navigator and outline to AI chat pages. Jump to prompts, preview nearby messages, and keep context in long chats.

### Description
Long conversations in ChatGPT and other AI tools quickly become hard to navigate. Finding a specific prompt or retracing context usually means endless scrolling and broken focus.

ChatDot adds a lightweight floating sidebar and message outline panel to supported AI chat pages, so you can move through long threads the way you scan a document outline.

Key features:
- Floating outline panel that extracts your prompts into a clickable conversation index.
- Quick navigation buttons for top, bottom, previous, and next user messages.
- Hover previews that show nearby prompt content before you jump.
- Highlight feedback after navigation so you can find the target message instantly.
- Light, dark, and system theme modes for a native-looking experience.
- Local-only processing with no cloud sync, telemetry, or remote data upload.

Supported platforms:
ChatGPT, Gemini, Claude, and Doubao, with more platforms being added over time.

## 日本語

### Summary
AI チャットページに右側ナビゲーターとメッセージ一覧を追加。任意の質問へすばやく移動し、長い会話でも流れを見失いません。

### Description
ChatGPT などの AI チャットで会話が長くなると、特定の質問や前後の文脈を探すために何度もスクロールする必要があります。

ChatDot は対応する AI チャットページに軽量な右側ナビゲーションバーとメッセージアウトラインを追加し、文書の目次を見るように会話全体をたどれるようにします。

主な機能:
- ユーザーの質問を抽出して一覧化するフローティングアウトラインパネル
- 上端 / 下端 / 前の質問 / 次の質問へ移動できるショートカットナビゲーション
- ジャンプ前に内容を確認できるホバープレビュー
- 移動先をすぐ見つけられるハイライト表示
- Light / Dark / System テーマモード対応
- ローカル完結で動作し、クラウド同期や遠隔送信なし

対応プラットフォーム:
ChatGPT、Gemini、Claude、豆包（今後さらに対応プラットフォームを追加予定）

## 한국어

### Summary
AI 채팅 페이지에 우측 내비게이터와 메시지 개요를 추가합니다. 원하는 질문으로 바로 이동하고 긴 대화에서도 흐름을 놓치지 않습니다.

### Description
ChatGPT 같은 AI 채팅에서 대화가 길어지면 특정 질문이나 문맥을 다시 찾기 위해 끝없이 스크롤해야 합니다.

ChatDot은 지원되는 AI 채팅 페이지에 가벼운 우측 내비게이션 바와 메시지 개요 패널을 추가하여, 문서 목차처럼 긴 대화를 빠르게 훑고 원하는 위치로 바로 이동할 수 있게 해줍니다.

주요 기능:
- 사용자 질문을 추출해 클릭 가능한 인덱스로 보여주는 플로팅 개요 패널
- 맨 위 / 맨 아래 / 이전 질문 / 다음 질문으로 이동하는 빠른 내비게이션 버튼
- 이동 전에 내용을 미리 볼 수 있는 호버 프리뷰
- 이동 후 목표 메시지를 바로 찾을 수 있는 하이라이트 표시
- Light / Dark / System 테마 모드 지원
- 모든 처리가 로컬에서만 이루어지며 원격 업로드 없음

지원 플랫폼:
ChatGPT, Gemini, Claude, Doubao(더 많은 플랫폼을 순차적으로 추가 예정)
