# ChatDot 商店文案多语言版

## English

### Summary
Add a right-side navigator, outline, and ChatGPT conversation trim mode to AI chat pages. Jump faster and keep long chats responsive.

### Description
Paste-ready plain text:

Long conversations in ChatGPT and other AI tools quickly become hard to navigate. Finding a specific prompt or reconstructing the surrounding context usually means endless scrolling and broken focus.

ChatDot adds a lightweight floating sidebar and message outline panel to supported AI chat pages, so you can move through long threads the way you scan a document outline. On ChatGPT, it also adds a conversation trim mode that keeps only the latest N turns visible to reduce page weight in very long chats.

Key features:
  Floating outline panel: extracts your prompts into a clickable conversation index.
  Quick navigation: jump to the top, bottom, previous, or next user message.
  Hover previews: inspect nearby prompt content before you jump.
  ChatGPT conversation trim: manually trim, restore, or auto-trim long chats while keeping only the latest N visible turns.
  Highlight feedback: the target message is emphasized after navigation.
  Native-looking UI: supports light, dark, and system theme modes.
  Local-only processing: no cloud sync, telemetry, or remote data upload.

Supported platforms:
  ChatGPT, Gemini, Claude, and Doubao, with more platforms being added over time.

## 日本語

### Summary
AI チャットページに右側ナビゲーター、メッセージ一覧、ChatGPT 会話圧縮を追加。長い会話でもすばやく移動できます。

### Description
貼り付け用プレーンテキスト:

ChatGPT などの AI チャットで会話が長くなると、特定の質問や前後の文脈を探すために何度もスクロールする必要があります。

ChatDot は、対応する AI チャットページに軽量な右側ナビゲーションバーとメッセージアウトラインを追加し、文書の目次を見るように会話全体をたどれるようにします。ChatGPT では、最新 N 件だけを残す会話圧縮モードも利用できます。

主な機能:
  フローティングアウトラインパネル：ユーザーの質問を抽出して、クリックできる一覧にまとめます。
  クイックナビゲーション：上端 / 下端 / 前の質問 / 次の質問へすばやく移動できます。
  ホバープレビュー：ジャンプ前に周辺メッセージの内容を確認できます。
  ChatGPT 会話圧縮：手動圧縮、復元表示、必要に応じた自動圧縮に対応し、最新 N 件だけを表示します。
  ハイライト表示：移動先のメッセージをすぐ見つけられます。
  ネイティブな見た目：Light / Dark / System テーマモードに対応します。
  ローカル完結：クラウド同期や外部送信は行いません。

対応プラットフォーム:
  ChatGPT、Gemini、Claude、豆包（今後さらに対応プラットフォームを追加予定）

## 한국어

### Summary
AI 채팅 페이지에 우측 내비게이터, 메시지 개요, ChatGPT 대화 축약을 추가합니다. 긴 대화에서도 더 빠르게 이동할 수 있습니다.

### Description
붙여넣기용 일반 텍스트:

ChatGPT 같은 AI 채팅에서 대화가 길어지면 특정 질문이나 문맥을 다시 찾기 위해 끝없이 스크롤해야 합니다.

ChatDot은 지원되는 AI 채팅 페이지에 가벼운 우측 내비게이션 바와 메시지 개요 패널을 추가하여, 문서 목차처럼 긴 대화를 빠르게 훑고 원하는 위치로 바로 이동할 수 있게 해줍니다. ChatGPT에서는 최근 N개만 남기는 대화 축약 모드도 사용할 수 있습니다.

주요 기능:
  플로팅 개요 패널: 사용자 질문을 추출해 클릭 가능한 인덱스로 보여줍니다.
  빠른 내비게이션: 맨 위 / 맨 아래 / 이전 질문 / 다음 질문으로 이동할 수 있습니다.
  호버 프리뷰: 이동 전에 주변 메시지 내용을 미리 확인할 수 있습니다.
  ChatGPT 대화 축약: 수동 축약, 복원 표시, 선택형 자동 축약을 지원하며 최근 N개만 남깁니다.
  하이라이트 표시: 이동 후 목표 메시지를 바로 찾을 수 있습니다.
  자연스러운 UI: Light / Dark / System 테마 모드를 지원합니다.
  로컬 처리: 모든 처리가 기기 안에서만 이루어지며 원격 업로드가 없습니다.

지원 플랫폼:
  ChatGPT, Gemini, Claude, Doubao(더 많은 플랫폼을 순차적으로 추가 예정)
